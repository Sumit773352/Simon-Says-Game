Welcome to Simon Says!

Instructions:



  • In this game, the computer will light up a sequence of buttons.
  • Repeat the sequence exactly in the same order.
  • After each round, the computer will add one more button to the sequence.
  • This means you must remember all the previous buttons and the new one it adds.


Example:
Round 1 → Yellow(new)
Round 2 → Yellow(old), Red(new)
Round 3 → Yellow(old), Red(old), Green(new)
...and so on.

If you click the wrong button, the game ends!
